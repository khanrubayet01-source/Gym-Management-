Here is exactly how to perform the test safely and how to revert to your current system immediately after. 1. The "Safety First" Backup Before touching any settings, take a photo of your current Cloud Server Setting screen. Write down these values: Server Address (likely a local IP or 0.0.0.0). Server Port (likely 4370 or 8081). No Hardware Harm: Changing these settings only redirects the data traffic. It does not delete fingerprints, RFID cards, or your device firmware. 2. Connecting to Your New Cloud Bridge On the F22, go to Menu > COMM. > Cloud Server Setting and enter your new Render credentials: Server Mode: ADMS. Enable Domain Name: ON. Server Address: gym-management-ybgq.onrender.com (Do not add https://). Server Port: 443 (This is the secure port Render uses). HTTPS: ON. The Verification: Exit to the main clock screen. Wait about 30–60 seconds. Look for the Cloud Icon at the top. If it turns Blue or Green, the F22 has successfully "shaken hands" with your Render server.
